This project is a Java-based FAT32 file system reader that acts as a shell for navigating and interacting with a FAT32 disk image. It allows users to mount a FAT32 image, list directory contents, change directories, read file data, and display file or directory statistics using simple commands. The program parses the FAT32 file system structures directly, supporting short file names and handling attributes like hidden, system, and archive. Included scripts automate mounting, unmounting, compiling, and testing the reader.
